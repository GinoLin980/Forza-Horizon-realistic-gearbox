Setup Instructions

Requirements

  • A controller or wheel is necessary.

    • 1. Open Game

  • Open up Forza Horizon 4/5

    • 2. Settings

    • Ensure gear changing is set to manual
    • Set Q and E keys to shift down and up, respectively

    Data Output

    • Data Output: ON
    • IP Address: 127.0.0.1
    • Port: 8000

    For Microsoft Store Version

    1. Run Ping.and.Retroflect.Utility.exe and choose an unused IP and ping it(recommended IPs are listed in-app).
    2. Unused IPs will be listed at the bottom. Choose one and click Retroflect.
    3. Open the game and set the UDP Data Out IP to the IP you chose to run in Retroflect and Data Out Port to 8000.
    4. Open FHGearbox_MS_store.exe and have fun!
